# 🚍 Efficient Public Transport Scheduling using Optimization Techniques

## 📌 Overview

This project focuses on building an intelligent **public transport scheduling system** using **Machine Learning and Optimization techniques**.
It predicts passenger demand and dynamically allocates buses, drivers, and conductors to improve efficiency and reduce waiting time.

---

## 🎯 Objectives

* Predict passenger demand based on historical data
* Optimize allocation of buses and staff
* Reduce passenger waiting time
* Improve operational efficiency of transport systems
* Provide real-time scheduling insights

---

## 🧠 Key Features

* 📊 Passenger Demand Prediction (ML Model)
* 🚦 Traffic-aware scheduling
* 📅 Date & time-based dynamic planning
* 🚌 Smart bus allocation
* 👨‍✈️ Driver & conductor assignment
* 📈 Live dashboard insights
* ⚙️ Automated scheduling system

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Efficient-Public-Transport-Scheduling-using-Optimization-Techniques.git
cd Efficient-Public-Transport-Scheduling-using-Optimization-Techniques
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```

## 📊 How It Works

1. Load historical transport data
2. Preprocess and encode features
3. Train ML model to predict passengers
4. Use optimization logic to allocate:
