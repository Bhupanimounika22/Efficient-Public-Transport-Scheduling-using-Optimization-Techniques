📘 About This Project

This project is an AI-powered Transport Optimization System developed as part of our Senior Design Project (SDP). The system predicts passenger demand using machine learning and intelligently allocates buses to improve efficiency and reduce operational costs.

Traditional transport systems rely on manual planning, which often leads to overcrowding or underutilized buses. Our system solves this problem by using data-driven techniques to make smarter and faster decisions.

⚙️ Technologies & Algorithms

  • 📊 Machine Learning using XGBoost
  • 📈 Regression for demand prediction
  • ⚙️ Optimization logic for bus allocation
  • 🌐 Flask for web application
